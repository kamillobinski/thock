## [What's New](https://github.com/kamillobinski/thock/compare/1.22.0...1.23.0)

> Meet the soundpack registry.

### New Features

- **Soundpack Registry**
  Modes are gone, replaced by a dedicated [soundpack registry](https://github.com/kamillobinski/thock-soundpacks). Custom soundpacks now show up in Settings where you can browse your collection.

- **List Command**
  A new `ls` command lets you see all registered soundpacks straight from the CLI.

### Improvements

- **Permission Warnings in Menubar**
  Alert windows for missing accessibility permissions are gone. Instead, a warning appears directly in the menubar with a quick link to grant access and a link to the docs for more context.

---

Thock now has a documentation website. Check it out at [thockapp.com/docs](https://thockapp.com/docs/1.23.0).
